# VendorStore

A MERN stack storefront solution built for Nigerian Instagram and WhatsApp vendors. Instead of managing orders through chaotic DMs, sellers get a clean, professional online store where buyers can browse products, view prices, and order directly via WhatsApp.

## What It Does

- Each seller gets their own unique store link
- Buyers browse products organized by category
- One click opens WhatsApp with a pre-filled order message
- Individual product links for sharing on Instagram stories and posts
- Sellers manage their own products through a simple dashboard
- Store is live 24/7 online

## Tech Stack

- **Frontend** — React + Vite + CSS3
- **Backend** — Node.js + Express
- **Database** — Railway
- **Image Upload** — Cloudinary
- **Auth** — JWT + bcrypt
- **Hosting** — Netlify (frontend) + Render (backend)
